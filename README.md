# 🍲 Swadse – Homemade Food by Mom

Swadse is a homemade food website that brings fresh, healthy, and delicious home-cooked meals prepared with love by Mom. Our goal is to provide authentic homemade taste, quality ingredients, and comforting food straight from the kitchen to your table.

## 🌟 Features

- 🍛 Fresh homemade meals
- 📋 Daily/weekly menu
- 🛒 Easy food ordering
- 📍 Delivery information
- 📱 Mobile-friendly design
- ❤️ Cooked with love and care

## 🚀 About Swadse

Swadse is a platform dedicated to serving homemade food prepared in a hygienic kitchen using quality ingredients. Every meal is freshly cooked to bring the warmth and taste of home to our customers.

Whether you miss home-cooked meals or want healthy food options, Swadse is here to serve you.

## 🥗 Food Categories

- Indian Home Meals
- Lunch & Dinner
- Homemade Special Dishes
- Seasonal Foods

## 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript / React (update based on your project)
- Backend: Node.js / Firebase 

## 📦 Installation

Clone the repository:

```bash
git clone <your-repository-link>
```

Go to the project folder:

```bash
cd swadse
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```
